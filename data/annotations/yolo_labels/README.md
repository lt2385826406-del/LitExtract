# YOLO Detection Labels

This directory stores annotation files for the YOLOv11 detection model.

## Copyright Notice

**Training images are NOT included in this repository.** All images are sourced from copyright-protected academic paper PDFs.

Annotation format follows the YOLO standard (class_id x_center y_center width height), and can be obtained upon request for academic collaboration.

## Category Definitions

See category configuration files under `configs/yolo/`.
